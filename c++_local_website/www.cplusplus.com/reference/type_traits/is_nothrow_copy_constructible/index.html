std::is_nothrow_copy_constructible

template <class T> struct is_nothrow_copy_constructible;
Is copy constructible throwing no exceptions
  • integral_constant
  • is_nothrow_copy_constructible

Trait class that identifies whether T is a copy constructible type, and such construction is known not to throw any exception.

This class inherits from integral_constant as being either true_type or false_type.

Template parameters

T
A complete type, or void (possible cv-qualified), or an array of unknown bound.

// is_nothrow_copy_constructible example
#include <iostream>
#include <type_traits>

struct A { };
struct B { B(const B&){} };
struct C { C(const C&) noexcept {} };

int main() {
  std::cout << std::boolalpha;
  std::cout << "is_nothrow_copy_constructible:" << std::endl;
  std::cout << "int: " << std::is_nothrow_copy_constructible<int>::value << std::endl;
  std::cout << "A: " << std::is_nothrow_copy_constructible<A>::value << std::endl;
  std::cout << "B: " << std::is_nothrow_copy_constructible<B>::value << std::endl;
  std::cout << "C: " << std::is_nothrow_copy_constructible<C>::value << std::endl;
  return 0;
}


Output:
is_nothrow_copy_constructible:
int: true
A: true
B: false
C: true
